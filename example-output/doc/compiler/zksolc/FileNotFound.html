FileNotFound

Short Description

No input file

Error Code: 2

Message: Can't find the file {path} to compile.

Fields

Field Name Type
path string
file_index uint

Description

The compiler is unable to find the file with the source code that it needs

Likely Causes

You did not provide a file to compile

Fixes:

  • Check if you have provided a correct Solidity or YUL file to compile.

Report:

Owner: solc (Version > 1.0.0)
